# AI Notes Summarizer

AI Notes Summarizer is a Flask-based web application that allows users to upload PDF, DOCX, and TXT files. The application extracts the text from the uploaded document and uses Google's Gemini AI to generate a concise summary.

## Features

- Upload PDF files
- Upload DOCX files
- Upload TXT files
- AI-generated summaries
- Clean and responsive user interface
- Flask backend

## Technologies Used

- Python
- Flask
- Google Gemini API
- PyPDF2
- python-docx
- HTML
- CSS
- Bootstrap
