# 🤖 AI Notes Summarizer

An AI-powered Notes Summarizer built with Flask and Google Gemini AI. Upload PDF, DOCX, or TXT files to generate intelligent summaries, quizzes, and downloadable study notes.

---

## ✨ Features

- 📄 Upload PDF, DOCX, and TXT files
- 🧠 AI-generated summaries
- 📌 Multiple summary modes:
  - Short Summary
  - Medium Summary
  - Detailed Summary
  - Exam Revision
- 🎯 AI-generated MCQ Quiz (up to 50 questions)
- ✅ Instant answer checking
- ❌ Wrong Answers Review after quiz
- 📋 Copy Summary
- 📥 Download Summary as PDF
- 📝 Download Summary as DOCX
- 🎨 Clean Bootstrap UI

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/mfahadsaleem001/AI-Notes-Summarizer.git
```

Move into the project folder:

```bash
cd AI-Notes-Summarizer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create:

```
models/config.py
```

Add your Gemini API key:

```python
GEMINI_API_KEY = "YOUR_API_KEY"
```

Run the project:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
